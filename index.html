Buka Tool

Panduan FB Prospek Generator

  • Masukkan API Key Gemini Anda (cukup sekali, otomatis tersimpan).
  • Pilih Mesin Robot AI (Rekomendasi: Gemini 2.5 Flash).
  • Masukkan Nama Produk / Bisnis Anda (Contoh: Nasi Bakar Mama Asha, MLM, dll).
  • (Opsional) Masukkan Target Spesifik jika ada (Contoh: Ibu Rumah Tangga).
  • Klik tombol Buat Strategi Prospek Sekarang.

Hasil: Anda akan mendapatkan target grup FB yang pas, 3 pemicu masalah (pain points), draf status pemancing rasa penasaran, serta template DM anti-spam.

Panduan AI Chat Generator

  • Paste Teks Chat / Pertanyaan dari calon pelanggan yang sulit dijawab.
  • Tuliskan Nama Produk & Fakta Bisnis Anda di kolom konteks agar AI paham.
  • Pilih Tujuan / Gaya Balasan (Handle Penolakan, Edukasi, Giring Closing, atau Follow Up).
  • Klik tombol Racik Balasan Chat Cerdas.

Hasil: AI menyajikan 3 Opsi Balasan (Singkat, Emosional, Profesional). Tinggal klik Copy dan kirim ke prospek!

Panduan AI Desain & Visual

  • Tuliskan Topik / Ide Postingan Anda (Contoh: PROMO STOK TERBATAS).
  • Pilih Gaya Visual Desain (Clickbait, Elegan, Emosional, Agamis).
  • Klik tombol Racik Konsep Desain.

Hasil: Anda mendapatkan racikan teks thumbnail CTR tinggi, rekomendasi visual Canva, skema warna, hingga prompt AI Bahasa Inggris untuk Bing Image Creator.

Panduan Konten Analyzer

  • Paste Draf Status FB / Tulisan Mentah yang ingin Anda posting.
  • Pastikan tulisan lumayan panjang agar AI bisa menganalisa strukturnya.
  • Klik tombol Analisa & Rombak Sekarang.

Hasil: Skor daya tarik konten (1-100), evaluasi hook & emosi, serta hasil tulisan ulang yang sudah dipoles dengan teknik copywriting AIDA (siap posting!).

Panduan SEO Keyword Tool

  • Tuliskan Topik / Nama Produk Anda.
  • Pilih Target Platform (FB Organic, YouTube, TikTok, dll).
  • (Opsional) Isikan Target Audiens Anda.
  • Klik tombol Riset Keyword Sekarang.

Hasil: 3 rekomendasi judul SEO, daftar kata kunci turunan (LSI), hashtag viral, dan deskripsi meta singkat untuk memanjakan algoritma.
